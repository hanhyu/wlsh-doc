#### 组件容器

## 环境搭建1

```
第一步： docker build -f Dockerfile -t hanhyu/wlsh:v1 .

第二步： docker-compose up -d

第三步： 进入mysql客户端导入数据库文件

第四步： 进入mongodb客户端创建baseFrame数据库，并创建用户名与密码
```
